### 6.3.3 发送点对点消息指定标记位

func IMSendPeeerMsgWithFlag\(msg string\)/发送点对点消息指定标记位

json格式：

```
{
    "api_name":"string",
    "appmsgID":1,
    "recvUID":1,
    "msgType":1,
    "msgflag":1,
    "msgLen":1,
    "data:"string"
}
```

信息字段含义：

| 字段 | 含义 |
| :--- | :--- |
| api\_name | 调用API名称 |
| appmsgID | 消息ID |
| recvUID | 接受者用户ID |
| msgType | 消息类型 |
| msgflag | 好友验证标志 |
| msgLen | 消息长度 |
| data | 发送的数据 |
